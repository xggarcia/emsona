Para añadir una imagen de fondo:

1. Coloca tu imagen de fondo (por ejemplo, "music-background.jpg") en esta carpeta.
2. Asegúrate de que la imagen tenga un tamaño adecuado (recomendado: 1920x1080 o similar).
3. Formatos recomendados: JPG o PNG.

Ejemplo de imagen: 
- music-background.jpg
- concert-background.png
- vinyl-pattern.jpg
